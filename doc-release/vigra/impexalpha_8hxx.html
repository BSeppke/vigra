Detailed Description

alpha-channel image import and export functions

This module implements functions importImageAlpha() and exportImageAlpha() that read or write images with an associated alpha channel (also known as "transparency" in other contexts). As in module impex.hxx the matching implementation for any given datatype is selected by template meta code.
